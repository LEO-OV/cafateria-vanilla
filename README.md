# Vanilla Coffee

Landing page responsive para una cafetería artesanal, desarrollada con HTML, CSS y JavaScript. El proyecto presenta una interfaz visual cálida y minimalista orientada a mostrar el menú, el proceso de preparación, información de la marca, ubicación y contacto.

## Vista general

Vanilla Coffee es una maqueta web estática para una cafetería ficticia ubicada en Ciudad de México. El sitio está diseñado para comunicar una propuesta de café artesanal, panadería fresca y un ambiente cercano para los clientes.

El proyecto fue desarrollado con tecnologías web base, sin frameworks, priorizando una estructura clara, diseño responsive e interacciones simples mediante JavaScript.

## Características

- Diseño responsive adaptable a escritorio, tablet y móvil.
- Hero principal con imagen de fondo y llamadas a la acción.
- Menú dividido por categorías: bebidas calientes, bebidas frías y panadería.
- Carrusel de productos controlado con JavaScript.
- Sección de proceso de preparación del café.
- Sección informativa sobre la cafetería.
- Mapa de ubicación embebido.
- Formulario de contacto con respuesta visual al usuario.
- Menú móvil desplegable con overlay.
- Estilos personalizados mediante variables CSS.

## Tecnologías utilizadas

- HTML5
- CSS3
- JavaScript Vanilla

## Objetivo del proyecto

Este proyecto tiene como objetivo practicar la construcción de interfaces web estáticas con HTML, CSS y JavaScript, aplicando estructura semántica, diseño responsive, organización visual y manipulación básica del DOM.
